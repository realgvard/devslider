2. Callbacks

Variable Default Type Description
devBeforeInit false function Before initialization callback.
devAfterInit false function After initialization callback.
devBeforeSlide false function Before move callback.
devAfterSlide false function After move callback.
devOnPause false function Call this function while pressed pause.
devOnPlay false function Call this function while pressed play.
